Equatorial Coordinates:
RA is sexagesimal hours (HH MM SS.ss or HH:MM:SS.ss) or decimal degrees (DDD.dd)
dec is sexagesimal or decimal degrees (DD MM SS.ss, DD:MM:SS.ss or DDD.dd)

Date of observation:
MJD is a decimal number. (MJD=JD-2400000.5)
Date is Year Month Day. YYYY-MM-DD, YYYYMMDD, YYYY/MM/DD are all valid though YYYY MM DD is not.
Time is 24-hour, colon separated sexagesimal hours. (HH:MM:SS or HH:MM)

Declination DEC is a reserved keyword in MySQL, so if you want to include keyword DEC in any search you must back-quote it. E.g., AND `dec`>0.

Keywords with hyphens MySQL interprets a hyphen as the -ve operator so any reference to a FITS keyword containing hyphens needs to be back-quoted. E.g., AND `CAT-EQUI`=1950. Without the quotes, MySQL will try to find fields named CAT and EQUI which differ by 1950.
